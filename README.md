# MoneyJuan Blog

A content-driven blog platform built with Astro, focusing on delivering high-quality articles with optimal performance and SEO capabilities.

## Overview

This repository houses a modern static blog site leveraging Astro's content collections architecture. The platform is optimized for content creation, search engine visibility, and reader engagement through a streamlined publishing workflow.

## Key Features

**Content Management**
- MDX support for enhanced content authoring with embedded components
- Structured content collections with type-safe frontmatter validation
- URL-driven article search integrated into the site header

**Performance & SEO**
- Static site generation for optimal loading speeds
- Automatic sitemap generation for search engine crawling
- RSS feed support for content syndication
- Canonical URLs and OpenGraph metadata implementation
- Optimized image processing with Sharp

**Publishing Infrastructure**
- Markdown-based content authoring workflow
- Automated build and deployment pipeline
- Version-controlled content history

## Technology Stack

- **Framework**: Astro 5.x
- **Content**: MDX with Astro's content collections
- **Image Processing**: Sharp for optimized image delivery
- **Integrations**: Sitemap, RSS feed generation

## Content Strategy

The blog follows a structured content creation process with:
- Brand voice and style guidelines
- SEO-optimized article templates
- Internal and external linking strategies
- Quality assurance workflows
